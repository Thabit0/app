# v0.7 UI Redesign

هذه النسخة تحوّل البرنامج من شاشة تقنية بسيطة إلى واجهة تشغيل يومية أقرب للوحة نظام داخلية للدويش.

## ما تغير

- برنامج واحد فقط على الجهازين.
- شريط جانبي ثابت باسم Dawish Content Studio.
- لوحة تحكم رئيسية بكروت حالة.
- صفحات واضحة:
  - لوحة التحكم
  - المنشورات
  - التعديل الجماعي
  - مركز الاعتماد
  - المعاينة
  - جهاز المحل والمزامنة
  - الأرشيف والسجل
  - الإعدادات
- وضع الجهاز أصبح واضحًا:
  - جهاز الإدارة
  - جهاز المحل
- صفحة جهاز المحل تعرض تشغيل محطة النشر والحالات والـ heartbeat.
- المعاينة أصبحت أقرب لشاشة جوال لكل منصة.
- TikTok يبقى صورة فقط افتراضيًا وبدون صوت.
- Snapchat يبقى صورة فقط.
- لا يوجد API.

## ملاحظة

هذه نسخة إعادة ترتيب وتجربة استخدام. المنطق الأساسي لمجلد المزامنة والـ jobs لم يتغير.
