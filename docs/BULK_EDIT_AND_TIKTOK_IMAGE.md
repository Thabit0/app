# Bulk Edit + TikTok Image Mode

## التعديل الجماعي

تقدر تحدد صور كثيرة وتطبق عليها:

- نفس المنصات.
- نفس رابط الموقع.
- نفس الكابشن الأساسي.
- نفس الهاشتاقات.
- نفس وضع TikTok.
- نفس قواعد السناب.
- نفس الجدولة مع توزيع كل صورة كمنشور مستقل.

## TikTok

الأوضاع:

1. `ImageOnly` — ينشر/يجهز الصورة كصورة.
2. `ImageFirstSilentVideoFallback` — يحاول الصورة أولًا، ولو ما ضبط يجهز فيديو صامت كاحتياط.
3. `SilentVideoOnly` — فيديو صامت فقط.

الافتراضي في هذا المشروع: `ImageOnly`.
