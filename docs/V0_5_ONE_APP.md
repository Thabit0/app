# v0.5 One App Mode

هذه النسخة تلغي فكرة أن المستخدم يحتاج Manager و Agent كبرنامجين منفصلين.

البرنامج الواحد يحتوي على وضعين:

- وضع تجهيز المحتوى على لابتوبك.
- وضع محطة النشر على جهاز المحل.

يتم الربط بين الجهازين عبر مجلد مزامنة فقط مثل OneDrive/Google Drive/Syncthing.

## تشغيل جهاز المحل

افتح نفس البرنامج، اختر نفس مجلد `DawishSync`، ثم اضغط:

`تشغيل محطة النشر على هذا الجهاز`

سيعمل البرنامج كل 30 ثانية ويفحص المنشورات المستحقة.

## لا يوجد API

كل النقل يتم عبر ملفات:

- jobs
- status
- heartbeat
- logs
- errors

ولا يوجد أي اتصال API للمنصات أو Google Drive.
