# Dawish Content Studio v0.8.5 Simple

برنامج واحد لجهاز المدير وجهاز المحل.

## الاستخدام

### جهاز المدير
1. افتح البرنامج.
2. اختر: أنا المدير.
3. أدخل Worker URL و ADMIN_TOKEN.
4. ارفع الصور.
5. اكتب الكابشن.
6. اختر المنصات واللوكيشن.
7. حدد الوقت.
8. اضغط رفع وجدولة.

### جهاز المحل
1. افتح نفس البرنامج.
2. اختر: جهاز المحل.
3. أدخل Worker URL و SHOP_TOKEN.
4. اضغط تشغيل محطة المحل.

## Cloudflare
الكود موجود في:

cloudflare/worker.js
