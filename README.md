# Dawish Content Studio v0.8.2 — Cloudflare Easy Final Review

برنامج ويندوز واحد وسهل للدويش.

## الفكرة

- نفس البرنامج يشتغل على جهازك وجهاز المحل.
- جهازك: ترفع الصور، تكتب الكابشن، تحدد اللوكيشن والوقت، وتجدول.
- جهاز المحل: عليه حسابات Instagram / TikTok / Snapchat، ويسحب المنشورات من Cloudflare وقت النشر.
- Cloudflare للمزامنة والتخزين فقط.
- لا يوجد API لمنصات التواصل.
- TikTok صورة فقط وبدون صوت.
- Snapchat صورة فقط.

## المطلوب من Cloudflare

تم اعتماده:

- Worker: `dawish-content-api`
- D1 binding: `DB`
- R2 binding: `MEDIA`
- Secrets: `ADMIN_TOKEN`, `SHOP_TOKEN`

انسخ الكود الموجود في:

```text
cloudflare/worker.js
```

وضعه داخل Worker ثم Deploy.

## داخل البرنامج

### على جهازك

1. افتح البرنامج.
2. اختر: أنا المدير.
3. ضع Worker URL.
4. ضع ADMIN_TOKEN.
5. ارفع الصور.
6. اكتب الكابشن.
7. ضع لوكيشن Instagram مثل: مدينة الزلفي.
8. ضع لوكيشن TikTok مثل: عطارة الدويش.
9. اختر وقت 12 ساعة.
10. اضغط رفع وجدولة.

### على جهاز المحل

1. افتح نفس البرنامج.
2. اختر: هذا جهاز المحل.
3. ضع Worker URL.
4. ضع SHOP_TOKEN.
5. اضغط تشغيل محطة المحل.
6. الجهاز يسحب المنشورات المستحقة ويفتح صفحة مساعدة للنشر.

## مراجعة v0.8.2

- تم منع تكرار فتح نفس المنشور على جهاز المحل بعد فتح صفحة المساعدة.
- تم إضافة فحص ادعاءات طبية داخل Worker نفسه، وليس داخل البرنامج فقط.
- TikTok يبقى `image_only` من السيرفر حتى لو وصل طلب مختلف.
- Snapchat يبقى `image_only` من السيرفر حتى لو وصل طلب مختلف.


## مراجعة v0.8.2 النهائية

- تم تقليل حساسية فلتر الكلام الطبي حتى لا يرفض عبارات تسويقية آمنة مثل: مناسب للمناسبات.
- بقي منع الادعاءات الطبية الحقيقية داخل البرنامج وداخل Worker.
- تم تحسين عرض الوقت محليًا بصيغة 12 ساعة في جهاز المحل وصفحة المساعدة.
- هذه هي النسخة الموصى برفعها على GitHub بدل v0.8/v0.8.1.
