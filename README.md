# Dawish Content Studio v1.0

برنامج Windows بسيط لإدارة صور عطارة الدويش وجدولتها بين جهاز المدير وجهاز المحل عبر Cloudflare.

## طريقة العمل

- جهاز المدير: يستورد مجلد صور كامل، وكل صورة تتحول إلى منشور مستقل.
- تعديل جماعي: كابشن واحد، منصات، لوكيشن، وقت بداية، وفاصل زمني.
- جهاز المحل: يسحب المنشورات المستحقة ويفتح صفحة المساعدة للنشر.
- TikTok وSnapchat: صورة فقط وبدون صوت.
- لا توجد API لمنصات التواصل.

## Cloudflare

Bindings المطلوبة:

- D1: `DB`
- R2: `MEDIA`
- Secret: `ADMIN_TOKEN`
- Secret: `SHOP_TOKEN`

انسخ `cloudflare/worker.js` إلى Worker ثم Deploy.

## البناء

ارفع محتويات المشروع إلى جذر GitHub. سيعمل Workflow باسم **Build Program** ويخرج Artifact جاهز لويندوز.

## اختصارات

- `Ctrl + O`: اختيار مجلد الصور
- `Ctrl + A`: تحديد الكل
- `Delete`: حذف المحدد من قائمة العمل
